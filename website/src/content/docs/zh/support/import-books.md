## 支持格式

ReadAny 支持以下常见电子书格式：

| 格式 | 扩展名 | 说明 |
|---|---|---|
| EPUB | `.epub` | 支持 EPUB 2 / EPUB 3 |
| PDF | `.pdf` | 更适合文字型 PDF |
| MOBI | `.mobi` | 旧版 Kindle 格式 |
| AZW / AZW3 | `.azw`, `.azw3` | 仅支持无 DRM 文件 |
| FB2 / FBZ | `.fb2`, `.fbz` | FictionBook 格式 |
| CBZ | `.cbz` | 漫画压缩包 |

## 导入后会发生什么

当你导入一本书时，ReadAny 通常会：

- 把书籍复制到应用管理的资料库目录
- 尝试提取标题、作者、封面等元数据
- 在本地数据库里创建书籍记录
- 保留原始文件不动

## 导入方式

### 拖拽导入

把一本或多本书直接拖进书库窗口。

### 导入按钮

1. 点击书库中的导入按钮
2. 选择一个或多个文件
3. 等待导入完成

### 文件关联

在桌面端，你可以让系统把支持的书籍格式默认交给 ReadAny 打开。

## 如果书是同步过来的

从另一台设备同步过来的书，有时会先只有元数据和封面，本地正文还没下载。

- 在桌面端，这类书会显示“需下载”状态。
- 点击打开时，会先下载正文文件，再进入阅读器。
- 所以“书已经出现在书库里”并不一定意味着正文已经在本地。

## 书库管理

- 按标题或作者搜索
- 按标题、作者、最近阅读、导入时间排序
- 用标签管理书籍
- 从书库中删除书籍

## 小建议

- 第一次使用时，先导入少量书籍，再去配置同步。
- 如果桌面端书很多，建议考虑在 **设置 → 通用** 里调整资料库目录。
- ReadAny 不支持 DRM 保护书籍。
