ReadAny 是一款本地优先的阅读器，支持 EPUB、PDF、MOBI、AZW、AZW3、FB2、CBZ 等常见电子书格式，并集成了笔记、高亮、翻译、AI 对话和语义搜索。

## ReadAny 适合做什么

- 统一管理你的 DRM-free 电子书
- 在阅读中做高亮、写 Markdown 笔记
- 结合 AI 聊天和翻译理解内容
- 对书籍做向量化和语义搜索
- 在多设备之间同步书库和阅读数据
- 通过局域网快传把一台设备的数据快速迁移到另一台

## 5 分钟上手

1. 安装 ReadAny
2. 导入几本书到书库
3. 打开一本书，试试高亮、笔记和搜索
4. 如果需要 AI，再去 **设置 → AI** 配置端点
5. 如果有多设备，再去 **设置 → 同步** 配置同步

## 平台说明

| 平台 | 说明 |
|---|---|
| macOS / Windows / Linux | 功能最完整，支持桌面端阅读器和资料库目录设置 |
| Android / iOS | 支持移动阅读、AI、同步和局域网导入 |

## 推荐顺序

如果你第一次使用，建议按这个顺序来：

1. 先装好应用
2. 先导入少量书籍
3. 先读书并试试高亮和笔记
4. 再配置一个 AI 端点
5. 最后再配置同步

## 一开始不用急着做的事

- 不配置 AI 也能正常阅读
- 单设备使用时，不需要先配置同步
- 不做语义搜索时，也不用先向量化
