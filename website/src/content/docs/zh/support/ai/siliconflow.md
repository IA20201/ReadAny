[硅基流动（SiliconFlow）](https://siliconflow.cn/) 是国内领先的 AI 模型服务平台，提供丰富的开源模型 API，包括对话模型和 Embedding 向量模型。

## 为什么选择硅基流动？

- **价格实惠**：按量计费，新用户有免费额度
- **模型丰富**：支持 DeepSeek、Qwen、Llama 等主流开源模型
- **国内访问**：服务器在国内，访问速度快
- **向量模型**：提供多种 Embedding 模型，可用于语义搜索

## 注册账号

1. 访问 [硅基流动官网](https://cloud.siliconflow.cn/)
2. 点击「注册」完成账号注册
3. 完成实名认证（部分模型需要）

## 获取 API Key

1. 登录后进入 [API 密钥管理](https://cloud.siliconflow.cn/account/ak)
2. 点击「新建 API 密钥」
3. 复制生成的 API Key（以 `sk-` 开头）

:::warning
请妥善保管 API Key，不要泄露给他人。
:::

## 配置 AI 对话

### 步骤

1. 进入 **设置 → AI**
2. 点击「添加端点」
3. 选择提供商为 **SiliconFlow**
4. 输入 API Key
5. 点击「获取模型」拉取可用模型列表
6. 选择要使用的模型

### 推荐模型

| 模型 | 说明 |
|---|---|
| **DeepSeek-V3** | 综合能力强，适合日常对话 |
| **DeepSeek-R1** | 推理能力强，适合复杂分析 |
| **Qwen2.5-72B-Instruct** | 阿里开源，中文能力强 |
| **Llama-3.3-70B-Instruct** | Meta 开源，英文能力强 |

## 配置向量模型（语义搜索）

硅基流动提供多种 Embedding 模型，可用于书籍的语义搜索功能。

### 步骤

1. 进入 **设置 → 向量模型**
2. 选择「远程 API」
3. 选择提供商为 **SiliconFlow**
4. 输入 API Key
5. 选择 Embedding 模型

### 推荐向量模型

| 模型 | 最大长度 | 说明 |
|---|---|---|
| **BAAI/bge-m3** | 8192 | 多语言支持，推荐中文用户使用 |
| **BAAI/bge-large-zh-v1.5** | 512 | 中文专用，速度快 |
| **BAAI/bge-large-en-v1.5** | 512 | 英文专用 |
| **Qwen/Qwen3-Embedding** | 32768 | 阿里开源，超长文本支持 |

### 使用向量模型

配置完成后：

1. 打开一本书
2. 点击侧边栏的「向量化」按钮
3. 选择「使用远程 API」
4. 等待处理完成

向量数据会发送到硅基流动服务器进行处理，处理完成后索引保存在本地。

:::info
移动端仅支持远程向量 API，不支持本地向量模型（为了减小应用体积）。
:::

## 费用说明

硅基流动采用按量计费：

- **对话模型**：按输入/输出 token 数量计费
- **向量模型**：按处理 token 数量计费

新用户注册后可获得免费额度，具体以官网为准。

## 错误排查

### API Key 无效

**错误表现**：提示「API Key 无效」或「认证失败」

**解决方案**：
- 确认 API Key 是否正确复制，没有多余空格
- 检查是否已完成实名认证
- 确认账户余额是否充足
- 尝试重新创建一个新的 API Key

### 模型列表为空

**错误表现**：点击「获取模型」后列表为空或报错

**解决方案**：
- 检查网络连接是否正常
- 确认 API Key 是否有效
- 部分模型需要先在硅基流动控制台开通
- 尝试手动输入模型名称

### 向量化失败

**错误表现**：向量化过程中断或报错

**解决方案**：
- 检查向量模型是否正确配置
- 确认账户余额是否充足
- 大文件可能需要较长时间处理，请耐心等待
- 检查网络连接是否稳定
- 尝试切换其他 Embedding 模型

### 连接超时

**错误表现**：请求超时或无响应

**解决方案**：
- 检查网络连接
- 如果在海外，可能需要使用代理
- 检查硅基流动服务是否正常（查看官网公告）
- 尝试更换网络环境

### 模型无响应

**错误表现**：发送消息后长时间无回复

**解决方案**：
- 检查模型是否在当前区域可用
- 确认模型名称是否正确
- 尝试切换到其他模型
- 检查账户是否有足够余额

### 余额不足

**错误表现**：提示「余额不足」或「额度用尽」

**解决方案**：
- 登录硅基流动控制台查看余额
- 充值账户
- 检查是否有免费额度未领取

### 请求频率限制

**错误表现**：提示「请求过于频繁」或「Rate Limit」

**解决方案**：
- 降低请求频率
- 等待一段时间后重试
- 升级账户等级获取更高限额
